A walkthough of the collaberative filtering method

In this blog post we will implement a machine learning algorithm to recommend games to users based on the game they like owe and like. The data is from www.arcadetrail.com — check it out, its awesome.

Technically, the type of recommender system we will implement is called collaborative filtering. We will construct at Matrix with the shape \( (# users)x(# games) \). This matrix contains all the votes and game ownerships for each user. The matrix is incomplete in the sense that not every user will have voted for every game. The aim is to establish which games a user might like, that is, we will fill out the missing parts of the \(R\) matrix. $$ \theta X = R^* $$ We do this by finding the matrixes \( X \) and \( \Theta \) whose inner product gives the filled out matrix \( R^* \).

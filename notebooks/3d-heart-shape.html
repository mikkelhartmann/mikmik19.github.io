Creating a 3D heart¶

I wanted to mare sure that I could recreate the 3D heart shape presented in Making a Heart with Mathematics before trying to to it in Three.js.

I import the libaries I will need.

In [1]:
import numpy as np
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D

I create the x,y, and z matrices that that define the heart.

In [2]:
u = np.linspace(0, 2 * np.pi, 100)
v = np.linspace(0, np.pi, 100)

x = 2*np.outer(np.cos(u), np.sin(v))
y = 1.2*np.outer(np.sin(u), np.sin(v)) - np.abs(x)*np.sqrt(((20-np.abs(x))/15))
z = np.outer(np.ones(np.size(u)), np.cos(v))*(2-np.divide(y,15))

I create a figure and add a 3d axes. The Axes3DSubplot.plot_surface() method lets me specify rcount, ccount that controlls the the number of faces the shape will have. I like a low polygon look.

In [3]:
fig = plt.figure(figsize=(5,4))
ax = fig.add_subplot(111, projection='3d')

ax.plot_surface(
    x, y, z,
    rcount=15, ccount=15,
    color='r', linewidth=0.5, alpha=1,
    shade=True
)

lim_min, lim_max = -1.4, 1.4

# I hide all the chart-junk because I'm only interested in the shape
ax.set_xlim([lim_min, lim_max])
ax.set_ylim([lim_min, lim_max])
ax.set_zlim([-.7, lim_max])

# Get rid of the panes
ax.w_xaxis.set_pane_color((1.0, 1.0, 1.0, 0.0))
ax.w_yaxis.set_pane_color((1.0, 1.0, 1.0, 0.0))
ax.w_zaxis.set_pane_color((1.0, 1.0, 1.0, 0.0))

# Get rid of the spines
ax.w_xaxis.line.set_color((1.0, 1.0, 1.0, 0.0))
ax.w_yaxis.line.set_color((1.0, 1.0, 1.0, 0.0))
ax.w_zaxis.line.set_color((1.0, 1.0, 1.0, 0.0))


ax.set_xticks([]) 
ax.set_yticks([]) 
ax.set_zticks([])

# Finally I set the camera angles.
ax.view_init(35, 90)

Finally the figure is saved as a svg file.
